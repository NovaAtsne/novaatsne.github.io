## 标题

正如上面的标题，我们将其称为“二级标题”，标题共有六个等级，“一级标题”最大，“六级标题”最小，以下为标题的基本教程

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

怎么样？你学会了吗？

## 文本部分

### 加粗

**文本加粗**是一个很常见的功能，主要使用`**文本**`或`__文本__`等语法实现。

### 斜体

*斜体*也是一个常见的功能，使用`*文本*`或`_文本_`实现，如果你想要 ___斜体加粗文本___ 话请使用`___文本___`语法

### 超链接

顾名思义，超链接是网站中那些颜色和别的字不一样的文字，点击后会跳转到另一个网址，我们将其称为“超链接”，使用以下代码实现

```markdown
[文本](链接)
```

如果你想要显示图片的话使用以下代码

```markdown
![文本](图片地址)
```

### 转义字符

在特殊字符前加反斜杠 `\` 使其显示为普通字符：  
`\* 这不是斜体 \*`

## 表格



Markdown 提供了一种简单的方式来创建表格。表格的基本语法是使用竖线 `|` 来分隔不同的单元格，使用连字符 `-` 来分隔表头和其他行。以下是一个基本的示例。



```markdown
| 表头1 | 表头2 |
| ----- | ----- |
| 单元格1 | 单元格2 |
| 单元格3 | 单元格4 |
```

### 对齐方式

Markdown 允许设置表格内容的对齐方式。可以通过在连字符的两侧添加冒号 *:* 来实现对齐。以下是一些示例：

- 左对齐：`:-`

- 右对齐：`-:`

- 居中对齐：`:-:`

## 脚注

使用以下代码创建脚注

```markdown
这是一个脚注示例[^1]。
[^1]: 这里是脚注的详细内容。
```

## 列表

列表是Markdown中非常基础的语法，列表主要包括无序列表与有序列表，无序列表是没有标号的列表，反之，有序列表则是有标号的列表，以下为列表实现方式：

### 无序列表

```markdown
- 列表1
- 列表2
    - 列表2.1
    - 列表2.2
- 列表3
```

### 有序列表

```markdown
1. 列表1
2. 列表2
    1. 列表2.1
    2. 列表2.2
3. 列表3
```

## 任务列表

用 `- [ ]` 或 `- [x]` 表示未完成/已完成：

```markdown
- [ ] 未完成任务
- [x] 已完成任务
```

## 代码

- **行内代码**：用一个反引号 ` 包裹  
  `print("Hello")`

- **代码块**：用三个反引号 ``` 包裹，可指定语言（语法高亮）：
  
  markdown
  
  ```python
  def hello():
      print("Hello World!")
  ```

## 分隔线

用三个或以上的 `-`、`*` 或 `_`（可含空格）：

```markdown
---
***
___
```

## 引用块

用 `>` 开头：

```markdown
> 这是一个引用块。
> 可以多行使用。
> 
> 引用内可嵌套其他Markdown语法。
```
