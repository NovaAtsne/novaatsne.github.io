教你如何用Hugo建立自己的博客

教你如何用Hugo搭建自己的博客
October 1, 2025

你是否看到网上那些主播都有自己的个人博客，你是否曾羡慕他们，自己也想拥有个博客？别急，今天，就让我教你如何建立自己的博客！现在，我将手把手教你使用Hugo免费建立自己的博客！

第一步：安装Hugo#

首先呢，要用Hugo建站肯定少不了安装Hugo啊，我们先点击这个神奇的 链接 下载Hugo，我比较建议下载最新版的extended版本，在里面找到适合你电脑运行的Hugo后下载下来。之后呢我们将下载好的Hugo解压并将里面的hugo.exe放到目录C:\hugo\bin\中。

然后我们还要将Hugo添加到PATH，如果你不这么做的话那么你在cmd或终端中运行任何hugo开头的命令都不会生效，这对于建立个人博客来说非常不好。那么应该如何将Hugo添加到PATH呢？不要急，听我慢慢说，首先我们要在 Windows 搜索栏中搜索 编辑系统环境变量 并打开它。在弹出的“系统属性”窗口中，点击右下角的“环境变量…”按钮。“环境变量”窗口的下半部分“系统变量”列表中，找到并选中名为 Path 的变量，然后点击 “编辑…”。在新建的行中，输入C:\hugo\bin\，这样我们就将Hugo添加到PATH中了

第二步：新建站点和安装主题#

在将Hugo添加到PATH后我们就要正式开始建站点咯，首先呢先按下你键盘中的Windows键和R键（Windows键就是你键盘上的那个窗户图标的键）并输入cmd，在里面按顺序输入以下命令

cd /d files # files换成你想将博客换成的目录 如果你是在非CMD环境下使用Hugo的话请将/d删除
hugo new site my-blog # my-blog可换成你的博客文件夹名称
cd my-blog

在输入好命令后我们还得要安装主题，这里呢我们以目前很流行的 PaperMod 作为实例。

首先呢，你必须要安装Git，否则话接下来的步骤就很难继续，你可以到 Git官网 下载Git，下载完后打开安装程序，一直点击Next就行了，之后你需要按照以下步骤安装主题：

  1. 进入你的 Hugo 项目目录
cd my-hugo-site
  1. 初始化 Git（如果还没有）
git init
  1. 添加主题作为 Submodule
# 这里以PaperMod作为示例，中间的链接部分可以换成你想要主题的Git链接
git submodule add https://github.com/adityatelange/hugo-PaperMod themes/ananke

在安装完主题后我们还需要告诉Hugo使用这个主题，因此，我们需要修改hugo.toml（通常是hugo.toml，但也有可能是hugo.yml，config.toml，config.yml等）文件

  1. 如果你的配置文件后缀是.toml的话，那么请将以下内容添加进配置文件
theme = "PaperMod"
  1. 如果你的配置文件后缀是.yml或.yaml的话，那么请将以下内容添加进配置文件
theme: "PaperMod"

至此，我们已将主题安装完毕

第三步：添加网站页面#

在我们安装好主题后，那么我们还缺什么呢？当然是我们的页面啦！没有页面你让读者访问个空气？那么，我将手把手教你在网站中建立你的第一个页面

建立主页（第一种）#

Hugo默认会以content/_index.md作为你网站的主页，如果你不介意内容不丰富的话，那么你可以尝试这个方法，这也是最简单的方法，首先我们需要在CMD/终端中输入以下内容：

hugo new _index.md

然后呢你打开网站根目录里的content文件夹，就会发现，欸，多了个my-name.md的文件。接下来，你只需要随便用一个软件打开这个文件，输入以下内容：

---
title: '主页'
date: 2023-10-20
---

...这里是你主页的内容...

之后你我们保存，然后呢我们应该如何看到他使用主题的样子呢？这我们就得再次在终端输入输入以下命令：

hugo server -D

之后你的终端应该出现类似于以下提示：

建立主页（第二种）#

建立主页和建立其他页面要特殊一些，首先我们要在网站根目录中的layouts文件夹中新建一个index.html文件，然后在这个文件中输入以下内容：

<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
</head>
<!-- style标签中可以放入你的网站主题代码 -->
<style>

</style>
<body>
    <header>
        <!-- #可以换成你想要换成的链接，你可以使用a标签制作多个导航栏 -->
        <a href="#">一个导航栏</a>
    </header>
    <h1>Hi，我叫 Find 1134</h1>
    <p>这是我的博客</p>
</body>
</html>

之后你修改完后将其保存我们的主页就制作好了

将网站部署到生产环境#

这是一个至关重要的步骤，不然的话你是想准备将你的博客文件打包发给观众并让观众解压后开启本地预览再看你的博客？那多麻烦啊，你何不将他弄到生产环境？这样的话观众们就不会这么麻烦了，那么应该怎么做呢？

首先，我们要打开我们的Github。然后呢注册一个账号，已经有账号的可以跳过这一步，之后我们点击那绿色按钮里的New，这样我们就会跳转到一个页面，请确保你的网页要和以下图片一样：

然后呢我们需要再次使用终端，依次在终端中输入以下命令：

hugo
cd public
git init
git branch -M main
git add .
git commit -m "更新我的网站"
git remote add origin https://github.com/【你的用户名】/【你的用户名】.github.io
git push -f origin main
cd ..
rm public -r -fo

之后，你在浏览器中输入https://【你的用户名】.github.io，就可以访问你的博客啦！你后面如果要修改你的博客的话，那么也只需要输入上面命令

当然，如果你像作者我一样非常懒的话，那么你就在你的博客更目录里创建一个名叫deplory.bat的文件。在里面输入以下内容：

@echo off
chcp 65001 >nul
echo 正在构建Hugo网站...
hugo

if errorlevel 1 (
    echo 错误：Hugo构建失败！
    pause
    exit /b 1
)

echo 正在部署到GitHub...
cd public

git init
git add .
git commit -m "更新网站"

echo 正在推送到GitHub...
git remote add origin https://github.com/Find1134/find1134.github.io.git
git branch -M main
git push -f origin main

cd ..
echo 清理临时文件...
rd /S /Q public

echo.
echo 网站部署完成！
pause

然后保存，之后呢每次更新完网站就直接双击打开这个文件就好了。
