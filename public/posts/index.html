警告！Windows 10寿命仅剩2天！微软将停更

众所周知，微软将于今年10月14日终止Windows 10的更新，用过停止支持的系统的人都知道，一旦系统停止支持了后，那么，黑客将盯上Windows 10这块大肥肉，因为目前Windows 10在全球仍是安装率最高的系统之一。不管你对黑客来说有没有价值，黑客的无障碍扫描都会盯上你的电脑，到时后果将不堪设想。眼前摆在用户面前有几条路可以走： 继续使用Windows 10。但不推荐这样做，因为若是选择继续使用Windows 10的话，那么你的电脑迟早会受到攻击。 升级Windows 11。这也是微软推荐大家这么做的，具体升级过程请看下文。 安装Windows 10LTSC。若不想升级Windows 11的话，安装Windows 10LTSC是一个不错的选择。 购买Windows 10ESU。购买Windows 10ESU的话需要花钱，不建议这么做。 使用Linux系统。这条路适合那些喜欢折腾的人，而转向Linux的话那么你将无法使用大多数Windows上的软件，因为Windows在如今已是全球最流行的系统之一，而大多数软件仅支持Windows和Mac甚至是仅支持Windows。 如何升级Windows 11 若你的电脑满足Windows 11升级条件的话，那么我们需要先到微软官网下载Windows 11，在官网中到下载适用于 x64 设备的 Windows 11 磁盘映像 (ISO)，选择Windows 11 专业中文版本，点击立即下载并按照接下来的步骤下载镜像。 在下载完镜像后，我们需要到Rufus官网下载Rufus写盘工具，在下载完Rufus写盘工具后我们打开它，选择你刚刚下载好的Windows 11 ISO镜像文件，选择好后点击开始，重要的一步来了在弹出的窗口中选择移除对TPM 2.0，4GB内存等要求选项（或类似选项），之后点击确定，在等待一会儿后，你就可以拔出U盘，接下里，你就可以为你想安装Windows 11的电脑插入这个U盘 我们在将Windows 11刻录到U盘后重启电脑，之后在开启电脑时一直按Del键（有些品牌的按键可能不同，但大多是F2，Del，F10，F11等）进入BIOS，在设置里按F7进入高级设置，进入Boot像，将UEFI开头的（也就是你的U盘）拖到第一个，之后按F10退出并保存，之后再重启，就会到Windows 11安装界面。到安装界面后你照着安装步骤即可安装完Windows 11！ 安装Windows 10LTSC 若你的电脑安装完Windows 11后非常卡顿或体验感非常糟糕的话，那么安装Windows 10LTSC是个不错的选择，Windows 10LTSC最长支持时间可达2032年！所以话不多说，Let’s go！ 首先我们要在MSDN官网注册新账号，然后登录完后点击里面的Windows 10字样，点击详细信息，划到最底下，选择LTSC，点击最上面的，点击复制，然后呢我们还需要一个BT下载工具，这里推荐qBittorrent，在qBittorrent里点击添加 Torrent 链接，在输入框中复制你刚刚复制的Windows 10LTSC链接，等待一会儿他就会提示下载成功。 在下载成功后我们再次拿出我们的U盘，依旧是打开Rufus，插入你的U盘，选择你的Windows 10LTSC镜像，将镜像刻录到U盘上后重启，将U盘设为第一启动项后我们就可以到Windows 10LTSC安装界面，Windows 10LTSC安装界面非常简洁，简单，适合那些喜欢简洁，不喜欢复杂的人群。在按照按照步骤按照完后我们就可你进入Windows 10LTSC系统里，我们可以看到Windows 10LTSC非常简洁，没有Microsoft Store等UWP应用，设备默认浏览器是IE。 好了，以上便是本文章所有内容，如果你喜欢的话，别只顾着白嫖啊，到我媒体上点个关注呗！Thanks~

教你如何用Hugo建立自己的博客

你是否看到网上那些主播都有自己的个人博客，你是否曾羡慕他们，自己也想拥有个博客？别急，今天，就让我教你如何建立自己的博客！现在，我将手把手教你使用Hugo免费建立自己的博客！ 第一步：安装Hugo 首先呢，要用Hugo建站肯定少不了安装Hugo啊，我们先点击这个神奇的 链接 下载Hugo，我比较建议下载最新版的extended版本，在里面找到适合你电脑运行的Hugo后下载下来。之后呢我们将下载好的Hugo解压并将里面的hugo.exe放到目录C:\hugo\bin\中。 然后我们还要将Hugo添加到PATH，如果你不这么做的话那么你在cmd或终端中运行任何hugo开头的命令都不会生效，这对于建立个人博客来说非常不好。那么应该如何将Hugo添加到PATH呢？不要急，听我慢慢说，首先我们要在 Windows 搜索栏中搜索 编辑系统环境变量 并打开它。在弹出的“系统属性”窗口中，点击右下角的“环境变量…”按钮。“环境变量”窗口的下半部分“系统变量”列表中，找到并选中名为 Path 的变量，然后点击 “编辑…”。在新建的行中，输入C:\hugo\bin\，这样我们就将Hugo添加到PATH中了 第二步：新建站点和安装主题 在将Hugo添加到PATH后我们就要正式开始建站点咯，首先呢先按下你键盘中的Windows键和R键（Windows键就是你键盘上的那个窗户图标的键）并输入cmd，在里面按顺序输入以下命令 cd /d files # files换成你想将博客换成的目录 如果你是在非CMD环境下使用Hugo的话请将/d删除 hugo new site my-blog # my-blog可换成你的博客文件夹名称 cd my-blog 在输入好命令后我们还得要安装主题，这里呢我们以目前很流行的 PaperMod 作为实例。 首先呢，你必须要安装Git，否则话接下来的步骤就很难继续，你可以到 Git官网 下载Git，下载完后打开安装程序，一直点击Next就行了，之后你需要按照以下步骤安装主题： 进入你的 Hugo 项目目录 cd my-hugo-site 初始化 Git（如果还没有） git init 添加主题作为 Submodule # 这里以PaperMod作为示例，中间的链接部分可以换成你想要主题的Git链接 git submodule add https://github.com/adityatelange/hugo-PaperMod themes/ananke 在安装完主题后我们还需要告诉Hugo使用这个主题，因此，我们需要修改hugo.toml（通常是hugo.toml，但也有可能是hugo.yml，config.toml，config.yml等）文件 如果你的配置文件后缀是.toml的话，那么请将以下内容添加进配置文件 theme = "PaperMod" 如果你的配置文件后缀是.yml或.yaml的话，那么请将以下内容添加进配置文件 theme: "PaperMod" 至此，我们已将主题安装完毕 第三步：添加网站页面 在我们安装好主题后，那么我们还缺什么呢？当然是我们的页面啦！没有页面你让读者访问个空气？那么，我将手把手教你在网站中建立你的第一个页面 建立主页（第一种） Hugo默认会以content/_index.md作为你网站的主页，如果你不介意内容不丰富的话，那么你可以尝试这个方法，这也是最简单的方法，首先我们需要在CMD/终端中输入以下内容： hugo new _index.md 然后呢你打开网站根目录里的content文件夹，就会发现，欸，多了个my-name.md的文件。接下来，你只需要随便用一个软件打开这个文件，输入以下内容： --- title: '主页' date: 2023-10-20 --- ...这里是你主页的内容... 之后你我们保存，然后呢我们应该如何看到他使用主题的样子呢？这我们就得再次在终端输入输入以下命令： ...
